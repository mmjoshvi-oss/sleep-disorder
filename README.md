# 😴 Sleep Disorder Prediction System

## 📌 Project Overview

The Sleep Disorder Prediction System is a Machine Learning web application that predicts the type of sleep disorder based on a person's health and lifestyle information. The system helps users understand potential sleep-related issues and encourages early awareness.

This application is developed using Python, Scikit-learn, and Streamlit.

---

## 🚀 Features

- 😴 Predict Sleep Disorder
- 🤖 Random Forest Machine Learning Model
- 📊 Interactive Streamlit Interface
- ⚡ Instant Predictions
- 💻 Easy-to-use User Interface
- ☁️ Deployable on Streamlit Community Cloud

---

## 🧠 Machine Learning Model

- Algorithm: Random Forest Classifier
- Programming Language: Python
- Framework: Scikit-learn

## 📊 Input Features

The model uses the following information:

- Gender
- Age
- Occupation
- Sleep Duration
- Quality of Sleep
- Physical Activity Level
- Stress Level
- BMI Category
- Blood Pressure
- Heart Rate
- Daily Steps

---

## 🎯 Output

The application predicts one of the following:

- No Sleep Disorder
- Insomnia
- Sleep Apnea
